# Telegram Comment Bot

Бот автоматически оставляет первый комментарий под каждым новым постом
в Telegram-канале (через группу обсуждения).

## Условия
- Канал привязан к группе обсуждения
- Бот — администратор в канале и группе

## Установка
pip install -r requirements.txt

## Запуск
python bot.py
